# noqqe.de

This is the source of [noqqe.de](https://noqqe.de).

# Build

Build an run the blog on `localhost:1313`

```
$ git clone https://github.com/noqqe/noqqe.de
$ cd noqqe.de
$ brew install hugo
$ hugo server
```

Just building works quite the same, ends up with the `documentroot` in
`public/`

```
$ cd noqqe.de
$ hugo
```
